# AtliQ Hotels - Hospitality Domain Analysis  
## Exploratory Data Analysis (EDA)

---

## Project Overview

This project presents an Exploratory Data Analysis of AtliQ Hotels' booking and revenue data across multiple Indian cities.  

The objective is to evaluate:

- Occupancy performance
- Revenue distribution
- City-level demand trends
- Weekend vs weekday behavior
- Customer rating patterns

The analysis supports data-driven decision-making in the hospitality domain.

## Key Insights

- Weekend occupancy (~72%) significantly higher than weekday (~50%)
- Delhi recorded highest average occupancy
- Mumbai generated highest total revenue
- Bangalore showed lowest average customer rating
- Premium and Presidential room categories showed stronger occupancy performance
